# Peccy-Protector-Tracker
🛡️ Peccy Protector Tracker
A web-based tool for tracking safety card distributions in Amazon's Peccy Protector Program at CMH4-FC.

Features
⚡ Log card distributions in under 10 seconds
🏆 AM & Associate leaderboards with shift-vs-shift scoreboard
🃏 16 collectible card types displayed in an interactive parade
📊 Real-time stats (total cards, active AMs, today's count)
📤 CSV export for reporting
🎖️ Peccy MVP monthly recognition info
📋 Built-in program instructions
How to Use
Open the site
Enter your AM login, the associate's login, and select the card type
Hit submit — it's logged to the shared leaderboard instantly
Tech
Single HTML file (no server required)
Google Sheets backend for shared data
Works on any browser — desktop or mobile
